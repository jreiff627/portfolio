Cow-Shaped PCB

Workshop Project - December 2025

A custom cow-shaped PCB built during an IEEE workshop. The board is fully functional and acts as a random number generator, lighting up two sets of 7 LEDs like rolling two six-sided dice. Each LED set corresponds to a die, and the extra LED in each set is used for formatting the dice faces. This project combined creativity with hands-on PCB design, soldering, and testing skills.

My Role

Designed the PCB layout, selected components, assembled, soldered, and tested the board to ensure functionality.

Skills / Knowledge Gained

  • PCB layout and design
  • Soldering and assembly techniques
  • Component placement and routing
  • Testing and troubleshooting circuits
  • Hands-on experience with creative electronics design

Resources Used

  • KiCad
  • Soldering tools and multimeter
  • Workshop lab equipment
