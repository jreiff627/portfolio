CPRE 2880 Final Project: RescueBot

December 2025

RescueBot is an assistive robotic navigation system designed to help visually impaired individuals navigate indoor spaces safely. It integrates real-time sensor feedback, audio interaction, and voice commands on an iRobot Create platform.

My Role

Designed and implemented the GUI for user interaction, enabling voice command input and real-time device feedback.
