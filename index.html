IEEE Workshop: Cow-Shaped Random Number Generator PCB

December 2025

A custom cow-shaped PCB built during an IEEE workshop. The board is fully functional and acts as a random number generator, lighting up two sets of 7 LEDs like rolling two six-sided dice. Each LED set corresponds to a die, and the extra LED in each set is used for formatting the dice faces. This project combined creativity with hands-on PCB design, soldering, and testing skills.

Learn More

CPRE 2880 Final Project: RescueBot

December 2024

A final project for CPRE 2880, exploring embedded systems and controls. RescueBot is an assistive robotic navigation system designed to help visually impaired individuals navigate indoor spaces safely. It integrates real-time sensor feedback, audio interaction, and voice commands on an iRobot Create platform.
