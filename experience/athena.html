AthenaGTX Internship

AthenaGTX · May 2025 – Aug 2025

During my internship at AthenaGTX, I worked on embedded firmware development for motion-tracking devices, collaborating closely with the hardware and mechanical teams to improve device functionality and optimize control systems.

Key Responsibilities

  • Developed embedded C firmware for device prototypes using BMI323 and BLE communication with BMD350 sensors.
  • Updated legacy firmware, implementing dual-color LED indicators and optimized pump control.
  • Created an Android app in Java to test BLE connectivity and resolve UI synchronization issues.
  • Collaborated on PCB design and produced technical documentation for internal and external stakeholders.

Skills Gained

  • Embedded C programming and firmware debugging
  • Sensor integration and hardware/software interaction
  • Cross-functional teamwork and technical documentation
  • Android app development for hardware testing

Project Outcome

Successfully delivered prototype firmware for the motion-tracking device and tested BLE communication with Android applications, contributing to the product development cycle and internal team efficiency.
