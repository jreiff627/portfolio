Undergraduate Research Assistant

Iowa State College of Engineering · Mar 2025 – Present

As an Undergraduate Research Assistant, I work on autonomous driving behaviors on the F1TENTH platform, programming miniature racecars with ROS2 for simulator and real-world deployment, testing control, planning, and sensor integration algorithms.

Responsibilities

  • Program autonomous driving behaviors on the F1TENTH platform using ROS2 for both simulation and real-world testing.
  • Design control and planning algorithms for obstacle avoidance, path following, and racing strategies.
  • Implement sensor fusion techniques using LiDAR, IMU, and cameras to improve localization accuracy.
  • Collaborate with team members to integrate hardware and software solutions for autonomous miniature racecars.

Skills Gained

  • C++ and Python programming for robotics applications
  • Sensor integration and fusion techniques
  • ROS2 framework for autonomous robotics
  • Control system design and testing in simulation and real-world environments

Project Outcomes

Developed and tested robust autonomous driving behaviors for the F1TENTH platform, improving racecar navigation and control under diverse conditions. Contributions enhanced lab research output and provided hands-on experience with real-world autonomous robotics systems.
